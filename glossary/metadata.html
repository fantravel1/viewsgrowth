Metadata

Definition

Metadata is all text information associated with a YouTube video: title, description, tags, captions, chapters, and category. Critical for SEO and discoverability.

Metadata Elements

  • Title: Most important—include primary keyword
  • Description: First 150 chars show in search; include keywords naturally
  • Tags: Help categorize (less impactful than title/description)
  • Captions: Searchable text; auto-generated or custom
  • Chapters: Timestamps for navigation and Key Moments
  • Category: Broad content classification

Metadata Priority

  1. Title (highest impact)
  2. Description (high impact)
  3. Captions/transcript (medium impact)
  4. Tags (low impact)

Best Practices

  • Front-load keywords in title and description
  • Write for humans first, algorithms second
  • Include call-to-action in description
  • Add links and timestamps
  • Update underperforming video metadata
